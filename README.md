# Secure Enterprise Network Architecture

## Overview

Designed and implemented a secure enterprise network architecture to protect organizational infrastructure from cyber threats. The project integrates multiple layers of security including firewall protection, intrusion detection systems, network segmentation, and secure remote access.

---

## Objectives

* Design a secure enterprise network architecture
* Implement firewall-based traffic control
* Detect threats using IDS
* Secure remote access using VPN
* Improve network monitoring and visibility

---

## Tools Used

* pfSense Firewall
* Snort IDS
* OpenVPN
* Kali Linux
* Wireshark

---

## Methodology

### 1. Network Design

* Created enterprise network structure
* Segmented network using VLANs

### 2. Firewall Implementation

* Deployed pfSense firewall
* Configured traffic filtering rules

### 3. Intrusion Detection

* Implemented Snort IDS
* Monitored suspicious activities

### 4. Secure Remote Access

* Configured OpenVPN
* Enabled encrypted communication

### 5. Monitoring & Logging

* Collected firewall logs
* Analyzed network traffic

---

## Key Features

* Firewall-based network protection
* Intrusion detection system (IDS)
* VLAN network segmentation
* Secure VPN access
* Real-time monitoring
